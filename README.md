## Formelsammlung für Latex4ei

Public Version: [latex4ei.de](http://latex4ei.de)

## Optional: Build with CMake
Download [UseLaTeX.cmake](https://cmake.org/Wiki/CMakeUserUseLATEX) and move to `/usr/share/cmake-X.X/Modules/.`

##### Steps to build:
```shell
mkdir build
cd build
cmake ..
make
```
